# IMDB-Sentiment-Analysis-Text-Classification

- Developed a sentiment analysis system on the IMDb dataset
- Performed text preprocessing (HTML removal, stopwords handling, lemmatization)
- Applied feature engineering using TF-IDF (n-grams) and Word2Vec embeddings
- Built and compared classification models (Logistic Regression, XGBoost)
- Evaluated models using cross-validation and standard metrics (Precision, Recall, F1-score)
